How to install CC 2018 +

I will show you how to install adobe premiere pro, the rest of the software will be installed in the same way
Step 1 : Turn off anti-virus software (for Windows 10)
Step 2 : Go to the download folder, extract it and click the set-up.exe file (note that you must extract it before installing and turning off the network) Step 3 : Click OK

Step 4 : Before Clicking Install, please uncheck the box Install hi.ru as homepage then press INSTALL

Note If you still leave a check mark in the middle box like in step 2, at this step Visual C++ redistributable will be installed automatically.

Step 5 : After completing the installation of Visual C++ redistributable, the software will switch to the default installation interface of Premiere. Click Continue to move to the next step.

At this step, the installation will be run automatically

Step 6 : Notice that the installation is completed, please click on the OK image or the X. So the installation is complete
